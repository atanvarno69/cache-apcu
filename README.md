## Requirements
**PHP >= 7.0** is required, but the latest stable version of PHP is recommended.

The [APCu](http://php.net/manual/en/book.apcu.php) PHP extension is required. 

## Installation
```bash
$ composer require atanvarno/cache-apcu:^0.1.0
```
